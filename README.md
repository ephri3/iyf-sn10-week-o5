# Week 5: DOM Manipulation - Interactive To-Do List

## Overview
This project demonstrates how to make web pages interactive using **JavaScript DOM manipulation**. Users can add, complete, and remove tasks dynamically. It includes exercises on selecting elements, handling events, and updating content and styles.

## Features
- Add tasks with a text input
- Mark tasks as completed
- Delete tasks individually or clear all
- Dynamic updates to the DOM without reloading the page
- Practicing event handling: click, input, submit, keyboard events

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- DOM API (selection, traversal, manipulation)
